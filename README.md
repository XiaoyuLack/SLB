# SLB
SLB 是 SlashBlade Resharped 的 NeoForge 1.21.1 附属模组，提供一个纯数据驱动的特殊效果（SE）系统。  你可以通过编辑 JSON 文件，为拔刀剑自定义被动效果——包括药水、额外伤害、点燃、闪电、吸血、二连击、虚空真实伤害、范围爆炸等 15 种原子效果类型，支持按等级施加惩罚。模组还提供 KubeJS 桥接层，让脚本可以读取刀的魂量、攻击力等属性，实现更复杂的进阶效果。  适合模组包作者和会改 JSON 的玩家。如果你只是想玩现成的刀，这个模组默认不附带任何成品刀和预置 SE——slb_ses.json 仅含一条示例，需要自行编辑或搭配资源包使用。
